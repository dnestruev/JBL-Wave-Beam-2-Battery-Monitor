# 🎧 JBL Wave Beam 2 Battery Monitor

Красивая анимация подключения и мониторинга заряда для наушников JBL Wave Beam 2.

![Demo](demo.gif)

## ✨ Особенности

- 📊 Отображение заряда каждого наушника и кейса
- 🎨 Тонкие цветные индикаторы
- 🔵 Синий цвет при зарядке
- 🔴 Красный мигающий при низком заряде
- 🎭 Полупрозрачные иконки наушников в кейсе
- 🚀 Плавная анимация появления/исчезновения
- 🛠️ Окно разработчика для тестирования

## 📦 Установка

```bash
git clone https://github.com/yourusername/JBL-Wave-Beam-2-Battery-Monitor.git
