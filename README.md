# 🎧 JBL Wave Beam 2 Battery Monitor

> **✨ БЕТА-ВЕРСИЯ** — уже сейчас можно тестировать анимацию и интерфейс!  
> **🚀 ПОЛНАЯ ВЕРСИЯ С BLUETOOTH ВЫЙДЕТ ЛЕТОМ 2026**

Красивая анимация подключения и мониторинга заряда для наушников **JBL Wave Beam 2**.  
Работает как на Mac — показывает заряд каждого наушника и кейса с плавными анимациями.

---

## 🎥 Демо

| Обычный режим | Зарядка | Низкий заряд |
|---------------|---------|--------------|
| 🟢 Зеленый 87% | 🔵 Синий | 🔴 Красный мигает |

---

## ✨ Что уже работает (Бета)

- 📊 **Визуализация заряда** — тонкие полоски 3px под каждым устройством
- 🎨 **Цветовая индикация**:
  - 🟢 Зеленый (>70%) — отлично
  - 🟡 Желтый (30-70%) — нормально
  - 🔴 Красный (<30%) — пора заряжать (мигает!)
  - 🔵 Синий — устройство заряжается
  - 💚 Светло-зеленый — 100% заряда
- 🎭 **Умные анимации**:
  - Наушник в кейсе → полупрозрачный + синяя полоска
  - Наушник вне кейса → яркий + цвет по заряду
  - Кейс на USB-C → синяя полоска
- 🚀 **Плавное появление/исчезновение** — как у AirPods
- 🛠️ **Окно разработчика** — можно тестировать любые сценарии

## 📥 Установка (Бета-версия)

### Простой способ:
1. Скачайте ZIP архив (Code → Download ZIP)
2. Распакуйте в любую папку
3. Установите Python с [python.org](https://python.org)
4. Запустите `setup.bat` (Windows) или `setup.sh` (Mac/Linux)
5. Дважды кликните `run.bat` или `run.sh`

### Через Git:
```bash
git clone https://github.com/dnestruev/JBL-Wave-Beam-2-Battery-Monitor.git
cd JBL-Wave-Beam-2-Battery-Monitor
pip install -r requirements.txt
python main.py
